pyrectbcm.rectangular_model module¶

pyrectbcm.rectangular_model.r_iteration(Input, l2, phij)[source]¶

Iterates the friction coefficient and yields flow velocities. Computes the flow solution in the inlets and basin, and iterates the velocity scale until it matches that found in the hydrodynamic solution (given a certain tolerance).

Parameters:
  • Input (Class) – Class containing all data for the various geograpical elements
  • l2 (np.ndarray) – 3d-array containing the L2-norm for the eigenfunctinos phi(m, n, j)
  • phij (np.ndarray) – 3d-array containing the integrated eigenfunctions over all j inlets
Returns:

flow velocity in the inlets ub (float): flow velocity in the basin mui2 (np.ndarray): frictional correction factor for the inlets mub2 (complex): frictional correction factor
Return type:

uj (np.ndarray)
